# Pythonメモ

## 1:演算

### 1-1:演算子について

下記のように演算子を使用します。

|     | 数学上の記号 | Python上の記号 |
| --- | ------ | ---------- |
| 加算  | +      | +          |
| 減算  | -      | -          |
| 乗算  | ×      | *          |
| 除算  | ÷      | /          |

他にもこのような演算子も存在します。

| 内容          | 記号  |
| ----------- | --- |
| 余りを求める      | %   |
| 商を整数で求める    | //  |
| べき乗（指数）を求める | **  |

### 1-3:代入の簡易記法

下記のように変数の代入を簡易化できます。

| 演算子 | 内容                            |
| --- | ----------------------------- |
| +=  | 自分自身に右辺の値を足して、その結果を自分自身に代入する  |
| -=  | 自分自身から右辺の値を減じて、その結果を自分自身に代入する |
| *=  | 自分自身に右辺の値を掛けて、その結果を自分自身に代入する  |
| /=  | 自分自身を右辺の値で割って、その結果を自分自身に代入する  |
